This program(3dPosition.py) returns the 3D position of drone given a JSON file as input.

Logic: - It uses the path taken till now which is stored in JSON file.
       - Extracts relevant position and prints the x-y-z co-ordinates of the drone.

Usage: - python3 3dPosition.py JSONfileXXX.json
       - Output: [X,Y,Z] co-ordinates
